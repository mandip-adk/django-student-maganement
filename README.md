# Student Management System

A simple Student Management System built using Django and Django REST Framework.
This project was created for learning purposes to understand backend development,
REST APIs, database operations, and basic UI integration.

---

## Features

- Create, Read, Update, and Delete (CRUD) student records
- Django REST Framework API endpoints
- Admin panel for managing students
- Basic Bootstrap-based explanation UI for viewing student data
- SQLite database integration

---

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
- HTML, CSS, Bootstrap

---

## Project Structure

- `students/` – Student app (models, views, serializers)
- `templates/` – HTML templates for UI
- `db.sqlite3` – SQLite database
- `manage.py` – Django project manager
